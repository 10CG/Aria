    Responsibilities: Identity management, session handling, OAuth integration

  Module 2: project-management (core domain)
    Features covered: project creation and management
    Responsibilities: Project CRUD, member management, project settings

  Module 3: task-management (core domain)

    Key components: TaskService, AssignmentEngine, TaskRepository

  Module 4: integrations (integration module)
    Features covered: Slack and GitHub integrations

    Responsibilities: API routing, authentication, rate limiting, versioning
