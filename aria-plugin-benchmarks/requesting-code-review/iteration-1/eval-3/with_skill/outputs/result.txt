## Issue Categorization Framework

Code Reviewer applies the following severity classification:

| Category | Definition | Action Required |
|----------|------------|-----------------|
| **Critical** | Security vulnerabilities, data loss risks, spec violations | Must fix before any approval |
| **Important** | Performance issues, missing error handling, bad architecture | Should fix before merge |
| **Minor** | Style issues, naming, missing comments, formatting | Fix when convenient |
