```markdown
# OpenSpec: OAuth2 Social Login

## Summary
Add OAuth2 authentication support for GitHub and Google providers,
allowing users to sign in without a password.

## Motivation
- Reduce signup friction (no password required)
- Leverage trusted identity providers
- Industry-standard OAuth2 / PKCE flow

## Scope
- Backend: OAuth2 callback handlers, token exchange, user linking
- Frontend: Social login buttons, redirect flow
- Database: oauth_accounts table linking social IDs to users

## Out of Scope
- Apple Sign In (future iteration)
- Enterprise SSO (separate spec)

## Acceptance Criteria
- [ ] Users can sign in with GitHub OAuth2
- [ ] Users can sign in with Google OAuth2
- [ ] Existing accounts can be linked to social providers
- [ ] OAuth tokens stored securely (not in plaintext)
```
