**路径**: `shared/contracts/openapi/api.yaml`
**描述**: 定义了完整的 API 端点契约，包括所有 HTTP 方法和路径

**主要 API 端点**:
- `/auth/login` (POST) - 用户登录
- `/auth/register` (POST) - 用户注册
- `/tasks` (GET, POST) - 获取任务列表 / 创建任务
- `/tasks/{id}` (GET, PUT, DELETE) - 获取/更新/删除特定任务

**代码段**:
