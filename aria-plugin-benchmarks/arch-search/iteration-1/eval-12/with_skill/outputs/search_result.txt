### 📝 信息摘要

API endpoints 主要通过以下方式定义：
1. **移动端**: 使用 Retrofit 注解定义 API 接口，位于 `@RestApi(baseUrl: '/api/v1')` 的抽象类中
2. **后端**: 使用 Flask Blueprint 和 `@route` 装饰器定义路由
3. **API 契约**: 标准化定义位于 `shared/contracts/openapi/` 目录

### 📁 代码文件指针

如需查看具体实现：
- `mobile/src/services/api/` - Retrofit API 服务接口定义
- `mobile/src/services/api/interceptors/` - 网络拦截器实现
- `backend/src/routes/` - Flask 路由定义（包含 tasks.py）
- `mobile/src/config/network_config.dart` - 网络配置
