## 技术栈总结

- **前端框架**: Flutter 3.x + Dart 3.x
- **状态管理**: Riverpod 2.0
- **网络**: Dio + Retrofit
- **存储**: Drift (SQLite) + flutter_secure_storage
- **路由**: go_router
- **AI**: LLM API 集成 + 向量存储

## 架构特点

1. **分层架构**: Presentation → Domain → Data → Infrastructure
2. **模块化设计**: 每个功能独立，通过文档导航
3. **离线优先**: 支持离线操作和自动同步
4. **安全优先**: JWT 认证 + 多重安全措施
5. **AI 增强**: 智能记忆系统和上下文感知
