### 📝 信息摘要

项目采用 JWT Token 认证机制，Mobile 端使用 AuthService 处理登录、登出和 Token 刷新。AccessToken 存储在内存中自动刷新，RefreshToken 使用 flutter_secure_storage 安全存储。Backend 基于 FastAPI 构建分层架构，包含认证服务层。安全措施包括 TLS 1.3 强制加密、SSL Pinning、代码混淆和 Root 检测。
