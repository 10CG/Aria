# Brainstorm 决策记录: Aria 2.0 US-020 范围与切片

## 背景

PRD v2.0 (Approved 2026-04-11) 列出 8 个待起草 User Story (US-020~027) 和 6 个 Milestone (M0~M6)，但 Story↔Milestone 映射不一一对应。本次 brainstorm 目标:

1. 锁定 US-020~027 的切片维度
2. 确定 M0 内高风险工作 (Hermes Spike, R1 法律确认) 的拆分粒度
3. 决定首个可执行 OpenSpec 的范围

## 锁定的决策

### D1: Story 切片维度 = 走法 C (混合)

| Story | 切片依据 | 原因 |
|---|---|---|
| US-020 (M0) | Milestone | M0 是前置验证, 不确定性高, Milestone Exit Criteria 即天然验收标准 |
| US-021 (M1) | Milestone | M1 是 MVP, PRD 已给出精确验收 (1 个 DEMO issue → PR) |
| US-022~027 (M2~M6) | 功能模块 | M2+ 功能并行度高, 按功能切灵活, 对应 PRD 原有 8 Story 的后 6 个 |

**否决的方案**:
- **走法 A (全按功能)**: 单 Story 跨多 Milestone, 验收时点模糊
- **走法 B (全按 Milestone)**: M2+ 阶段 Story 名退化为项目阶段, 失去用户能力语义

### D2: Hermes Spike = US-020 内 task, 不独立 Story

**否决方案 B (拆出 US-020a Hermes Spike)** 的原因:
- Spike timeboxed 1 sprint, 规模 (~40h) 未超过 US-020 总工时 (~80h) 的 50%
- Spike 已经是 PRD M0 必做项, 单独成 Story 增加管理成本
- Spike 反转架构的风险通过独立 OpenSpec (见 D4) 跟踪即可

### D3: R1 Anthropic 法律确认降级

**降级原因** (4 点判断):

1. `npm install -g @anthropic-ai/claude-code` 是公共包标准用法, 不构成"再分发"
2. Forgejo 是 10CG 私有 registry, 仅 Aether 集群拉取, 非公开分发
3. headless `claude -p` 用 10CG 自有 API key, ToS 允许程序化使用
4. 镜像名 `aria-runner:claude-latest` 私用不构成商标侵权

**新处理**:
- ❌ 不再联系 Anthropic support
- ✅ US-020 内增加 task: 调用 legal-advisor agent 内审 4 点 (~1h)
- ✅ R1 风险等级由 "M1 hard precondition" 降为 "已评估的低风险"

**保留逃生口**: 若 legal-advisor 内审发现新疑点, 再决定是否升级回邮件确认。

### D4: 首个 OpenSpec 切片 = 选项 C (总 Spec + Spike 子 Spec)

```
openspec/changes/
├── aria-2.0-m0-prerequisite/        ← 总 Spec, Level 3
│   ├── proposal.md
│   └── tasks.md   (R1 / A1 / A3 / AD 文档 / M0 Report)
└── aria-2.0-m0-spike-hermes/        ← Spike 单独 Spec, Level 3
    ├── proposal.md  (含 Spike 设计 + 评估指标 + 裁决条款)
    └── tasks.md     (Spike 实现 / LoC 量化 / rebase 成本评估 / 报告)
```

**否决方案**:
- **A (1 个大 Spec)**: tasks.md 过大, Spike 失败时整 Spec 难以处理
- **B (5 个小 Spec)**: 管理开销大, R1/A1/A3 是低风险调研无需独立 Spec

**两 Spec 协同**: Spike Spec 的"裁决结论"通过 OpenSpec 引用机制写回总 Spec 的 AD3 段落。

- **Acceptance**: PRD M0 Exit Criteria 三项 (AD 文档 + Spike 报告 + 产品负责人 Go/No-Go)
