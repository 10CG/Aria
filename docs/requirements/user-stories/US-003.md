# US-003: 多 AI 平台兼容性

> **Priority**: LOW
> **Status**: pending
> **Parent PRD**: prd-aria-v1.md
> **Target Version**: v1.2.0

---

## User Story

**作为** 使用不同 AI 工具的开发者，
**我希望** Aria 方法论不绑定特定 AI 平台，
**以便** 在 Claude Code 以外的 AI Agent 环境中也能应用。

## 验收标准

- [ ] 方法论核心 (十步循环、OpenSpec) 与 AI 平台无关
- [ ] 至少 1 个非 Claude 平台的概念验证
- [ ] 文档明确区分平台无关部分和平台特定部分
- [ ] 提供平台适配指南

## 技术备注

- standards/ 子模块应保持平台无关
- Skills/Agents 是平台特定实现，需要适配层
- 核心概念 (OpenSpec, 十步循环) 的 Markdown 格式天然跨平台
