## User Story

**作为** Aria 方法论的研究者，
**我希望** 在非 Aria 项目中验证方法论的适用性，
**以便** 证明 AI-DDD 方法论具有普适性和可重现性。

## 验收标准

- [x] 至少 1 个外部项目成功应用 Aria 十步循环 — Kairos (TypeScript/Node.js AI 销售自动化) 完成试点
- [x] 记录外部项目适配过程中的调整和问题 — ADAPTATION_LOG.md，结论：零适配需求
- [x] 外部项目的 Skill 效果提升与 Aria 项目基线数据可比 — 3 Skills x 4 evals, delta 可比
- [x] 输出跨项目适用性报告 — [aria-plugin#2](https://forgejo.10cg.pub/10CG/aria-plugin/issues/2)
